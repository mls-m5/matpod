# matpod

A version of software for the Vorpal hexapod.

## Build

### Install platformio

Install in vscode if you use that or: 
https://platformio.org/install

### Build from command line

cd into directory

```bash
platformio run
```

### Flash to esp

```bash
platformio run -t upload
```
